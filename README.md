
# Finance tracker

Представляет собой консольное приложение, предоставляющее возможности учёта финансовых операций.

## Возможности

- Добавление, редактирвоание и удаление операций
- Управление категориями
- Фильтр и сортировка
- Отображение баланса доходов/расходов
- Отображение последних N операций
- Резервное копирование и восстановление файла формата JSON

## Стек

- Pyhton 3.13 для работы с JSON, dataclass, typing и enum

## Структура

```
src/
├── console.py
├── finance_manager.py
├── storage.py
├── operation.py
├── category.py
└── main.py
```

## Запуск

python main.py

## Архитектура

Имеется 3 слоя:
- UI (Console)
- Бизнес-логика (FinanceManager)
- Обработка JSON данных (Storage)

Архитектура выстроена так, чтобы бизнес-логику можно было вынести отдельно, присоединив другой способ отображения UI (например, GUI):
Console -> FinanceManager -> Storage
